# DevOps Project

A full-stack application with backend and frontend components.

## Project Structure

```
.
├── backend/          # Node.js backend server
│   ├── app.js
│   └── package.json
├── frontend/         # Frontend application
└── README.md
```

## Getting Started

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```

## Requirements

- Node.js (v14 or higher)
- npm or yarn
